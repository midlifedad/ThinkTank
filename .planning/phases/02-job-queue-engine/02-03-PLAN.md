    class JobHandler(Protocol):
        """Protocol for job handlers. Every handler must implement this.

        Handlers receive an AsyncSession and the Job being processed.
        They should raise on failure (caught and categorized by the worker loop).
        The session is a fresh session -- NOT the claim session.
        """

    `src/thinktank/handlers/registry.py`:
    ```python
    from src.thinktank.handlers.base import JobHandler

    # Registry populated by Phase 3+ as handlers are implemented.
    # Key: job_type string, Value: callable matching JobHandler protocol.
    JOB_HANDLERS: dict[str, JobHandler] = {}

    def register_handler(job_type: str, handler: JobHandler) -> None:
        """Register a handler for a job type. Raises ValueError if already registered."""
        if job_type in JOB_HANDLERS:
            raise ValueError(f"Handler already registered for job type: {job_type}")
        JOB_HANDLERS[job_type] = handler

    def get_handler(job_type: str) -> JobHandler | None:
        """Get the handler for a job type. Returns None if not registered."""
        return JOB_HANDLERS.get(job_type)
    ```

    `worker_loop(session_factory: async_sessionmaker, settings: WorkerSettings | None = None) -> None`:
    The main async worker loop. Structure:

    1. Initialize: shutdown_event (asyncio.Event), semaphore (asyncio.Semaphore(settings.max_concurrency)), active_tasks set, worker_id via generate_worker_id
    2. Register SIGTERM and SIGINT signal handlers to set shutdown_event
    3. Start reclamation scheduler: asyncio.create_task that runs reclaim_stale_jobs every settings.reclaim_interval seconds
    4. Main loop (while not shutdown_event.is_set()):
       a. Check kill switch: open session, call is_workers_active(). If False, sleep poll_interval, continue.
       b. Claim job: open session, call claim_job(session, worker_id, settings.job_types). If None, increment idle_count, sleep with backoff (min(poll_interval * idle_backoff_multiplier^idle_count, max_idle_backoff)), continue.
       c. Reset idle_count on successful claim.
       d. Dispatch: acquire semaphore, create_task for _process_job(), add to active_tasks, add done_callback to discard from active_tasks.
    5. Graceful shutdown: cancel reclamation task, wait for active_tasks with timeout=60.

    `_process_job(session_factory: async_sessionmaker, job: Job, semaphore: asyncio.Semaphore, worker_id: str) -> None`:
    Internal function for processing a single job:
    1. Look up handler via get_handler(job.job_type)
    2. If no handler: open session, call fail_job with error_category=ErrorCategory.HANDLER_NOT_FOUND, release semaphore, return
    3. Open fresh session for handler execution
    4. Try: call handler(session, job)
    5. On success: open session, call complete_job(session, job.id)
    6. On exception: categorize via categorize_error(exc), open session, call fail_job with error message and category
    7. Always release semaphore in finally block
    8. Log job lifecycle events via structlog (job_claimed, job_completed, job_failed, handler_not_found)

    `_reclamation_scheduler(session_factory: async_sessionmaker, interval: float, shutdown_event: asyncio.Event) -> None`:
    Internal async function:
    - while not shutdown_event.is_set():
    -   try: open session, call reclaim_stale_jobs, log results
    -   except: log error, continue
    -   await asyncio.sleep(interval) or shutdown_event.wait with timeout
