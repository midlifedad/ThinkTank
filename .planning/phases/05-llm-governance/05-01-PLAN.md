       - Method `async def review(self, system_prompt: str, user_prompt: str, response_schema: type[BaseModel], max_tokens: int = 4096) -> tuple[BaseModel, int, int]`:
         - Records `time.monotonic()` start.
         - Calls `self._client.messages.create()` with model, max_tokens, system prompt, user message, and `tools` parameter containing a single tool definition derived from the Pydantic schema's JSON schema via `response_schema.model_json_schema()`.          - Computes `tokens_used = response.usage.input_tokens + response.usage.output_tokens`.
         - Returns `(parsed_result, tokens_used, duration_ms)`.
       - NOTE: Use tool_use approach for structured output (create a tool with the Pydantic JSON schema, force tool_choice={"type": "tool", "name": "structured_output"}). This is the reliable approach that works across all SDK versions. Do NOT use `messages.parse()` or `output_format` -- these may not be available in the installed SDK version. The tool_use pattern is universally supported.
